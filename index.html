Scripting for Web and Mobile I

Preview About This course

Official Note

This course focuses on scripting for web and mobile applications, and is designed for students with no prior programming experience. JavaScript, markup, script libraries, and other tools and technologies will be used to create simple applications and prototypes.

Personal Note

There are tons of tidbits that you need to know about. All these little pieces of knowledge that it takes to make piece of the internet. The hardest part of this class will be creative execution, not the bits and pieces you need to become familar with. As proof, if you wanted to be a cabby in london you'd have to memorize 25,000 streets and any business or landmark on them. So yeah, in 15 weeks you will have the gist of Hypertext Markup Language, a bunch of Cascading Stylesheets and decent understanding of Javascript. The brain will fill, it will become sore, you may become frustrated but you are going to do well.
